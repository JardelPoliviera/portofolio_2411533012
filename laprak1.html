1. Tujuan Praktikum

Tujuan utama dari praktikum ini adalah agar mahasiswa memiliki kemampuan teknis dalam melakukan instalasi framework Laravel serta mampu membuat proyek baru menggunakan framework tersebut. Selain aspek teknis instalasi, mahasiswa diharapkan dapat mengenal struktur direktori yang ada di dalam Laravel dan memahami secara mendalam mengenai konsep arsitektur MVC (Model-View-Controller) yang menjadi landasan utama pengembangan aplikasi dengan Laravel.

2. Hasil & Pembahasan

1. Install XAMPP : Download XAMPP pada link https://www.apachefriends.org/index.html kemudian install sesuai dengan Langkah-langkah wizard, setelah berhasil diiinstall maka akan muncul folder htdocs secara default pada C:\xampp\htdocs yang nantinya akan digunakan untuk membuat project-project Laravel. Dengan menginstall XAMPP maka secara otomatis akan terinstall webserver apache, PHP dan PhpMyAdmin.

2. Install Composer : Composer merupakan package manager untuk PHP, composer akan digunakan untuk menambahkan package-package yang dibutuhkan pada saat development. Download compose pada link https://getcomposer.org/Composer-Setup.exe, selanjutnya install sesuai dengan Langkah-langkah wizard.

3. Install GIT : Silahkan download dan install GIT pada link berikut https://git-scm.com/downloads/win

4. Install Node JS dan NPM : Node JS pada Laravel berfungsi untuk menangani masalah frontedn dan build asset UI (Library UI). Buka situs resmi node js https://nodejs.org/ kemudian download dan install sesuai dengan Langkah-langkah wizard. NPM (Node Package Manager) yang berfungsi mengelola paket untuk ekosistem Javascript, NPM biasanya secara otomatis terinstall Ketika menginstall node js.

3. Kesimpulan

Laravel merupakan framework PHP open-source yang sangat populer untuk pengembangan aplikasi web berbasis arsitektur MVC. Untuk menjalankan Laravel 12, sistem harus memenuhi kriteria tertentu, seperti versi PHP minimal 8.2 dan berbagai ekstensi PHP pendukung lainnya. Proses pengembangannya didukung oleh ekosistem alat seperti Composer sebagai package manager, Node.js untuk pengelolaan aset frontend, serta Artisan CLI yang memudahkan otomatisasi tugas-tugas pengembangan. Dengan fitur-fitur modern seperti Eloquent ORM dan Blade Templating, Laravel memungkinkan pengembang untuk membangun aplikasi yang aman, skalabel, dan terorganisir dengan baik.

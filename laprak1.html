1. Tujuan Praktikum

Mahasiswa diharapkan mampu memahami konsep dasar HTML5, struktur dokumen, dan penggunaan tag-tag semantik untuk membangun kerangka website yang baik dan aksesibel.

2. Hasil & Pembahasan

Pada praktikum ini, saya telah berhasil mengimplementasikan struktur dasar website menggunakan tag <header>, <main>, dan <footer>. Berikut adalah contoh kode yang digunakan:

index.html 
<!DOCTYPE html>
<html>
  <head>
    <title>Latihan Dasar</title>
  </head>
  <body>
    <h1>Halo, ini laporan saya!</h1>
  </body>
</html>

Hasil tampilan dari kode di atas menunjukkan bahwa browser merender tag Heading 1 dengan ukuran font paling besar secara default...

3. Kesimpulan

Penggunaan elemen HTML yang tepat sangat krusial dalam pembuatan website. Memasukkan laporan langsung ke dalam web (seperti halaman ini) meningkatkan user experience dibandingkan hanya menggunakan format PDF.

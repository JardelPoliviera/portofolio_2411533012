DESKRIPSI SINGKAT:
Website ini adalah portofolio digital personal yang dibangun menggunakan HTML5, CSS3, dan framework Bootstrap 5.3.
Website ini terdiri dari 4 halaman utama yang responsif sepenuhnya.

TENTANG SAYA:
Website ini menceritakan perjalanan saya, Jardel Poliviera, sebagai mahasiswa Informatika Universitas Andalas, mencakup riwayat pendidikan, keahlian, dan karya-karya saya.

FITUR UTAMA:
1. Halaman Utama (Home): 
   - Sidebar Navigation yang responsif (berubah menjadi Navbar pada tampilan mobile).
   - Hero section dengan background transparan.
   - Carousel Slider untuk "Featured Highlights" (Github, LinkedIn, Web).
   - Grid layout untuk menampilkan skill.

2. Halaman About: 
   - Tampilan profil dengan gaya header modern.
   - Fitur Accordion (Menu lipat) untuk menampilkan detail Pendidikan, Pengalaman, dan Keahlian secara rapi dan interaktif.

3. Halaman Portfolio: 
   - Galeri proyek dengan fitur "Live Filtering" menggunakan JavaScript (Kategori: All, Websites, Designs, Apps).
   - Detail Modal (Pop-up) Bootstrap untuk melihat deskripsi lengkap proyek tanpa harus berpindah halaman.

4. Halaman Kontak: 
   - Formulir pesan interaktif dengan validasi input.
   - Informasi kontak yang disusun dengan layout kartu.
   - Embed Google Maps lokasi.

CARA MEMBUKA:
1. Pastikan perangkat terhubung ke internet (untuk memuat CDN Bootstrap, FontAwesome, dan Gambar).
2. Ekstrak folder project.
3. Buka file 'index.html' menggunakan browser modern (Chrome/Edge/Firefox).
4. Alternatif: Akses versi online (jika sudah di-deploy) melalui link: poliviera.ifportofolio.com
